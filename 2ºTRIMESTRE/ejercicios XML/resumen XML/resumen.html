Resumen de XML

¿Qué es XML?

XML (eXtensible Markup Language) es un lenguaje de marcado que define reglas para estructurar datos de manera legible tanto para humanos como para máquinas.

Características principales

  • Es autodescriptivo y fácil de entender.
  • Permite estructurar datos en jerarquías.
  • Es extensible, lo que significa que puedes crear tus propias etiquetas.
  • No depende de un software o plataforma específica.

Diferencias entre HTML y XML

  • Propósito: HTML muestra contenido, XML transporta datos.
  • Estructura: HTML usa etiquetas predefinidas, XML etiquetas personalizadas.
  • Flexibilidad: HTML es menos estricto, XML exige cierres correctos.
  • Enfoque: HTML diseña contenido, XML organiza datos.
  • Uso: HTML para la web, XML para sistemas y aplicaciones.

Documentos Bien Formados en XML

Cumplen las reglas sintácticas definidas por el estándar XML de W3C. Estas son las normas básicas:

  • Prólogo definido: Versión y codificación.
  • Estructura de árbol: Un único nodo raíz.
  • Etiquetas emparejadas: Apertura y cierre.
  • Atributos entre comillas: Simples o dobles.
  • Distingue mayúsculas de minúsculas.
  • Nombres de etiquetas: Similares a variables en programación.
  • Comentarios: Fuera de etiquetas con <!-- comentario -->.

Ejemplo básico de XML

<libro>
    <titulo>Aprendiendo XML</titulo>
    <autor>Jesús Panadero</autor>
    <año>2025</año>
</libro>
